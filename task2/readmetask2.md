Overview

This project uses an Arduino Uno to control a servo motor with a potentiometer (knob). safety feature in the code that stops when past 68 degrees. An LED turns on to warn you when this limit is reached.


Components Used

    1x Arduino Uno

    1x Servo Motor

    1x Potentiometer 

    1x LED (Red)

    1x 220 Ω Resistor

    Breadboard and connection  wires

Wiring

    Potentiometer: Middle pin to A0.

    Servo: Signal wire to Pin 9.

    LED: Long leg to Pin 13, short leg to resistor then GND.


code    

    The code is written in C++ using the Servo.h library
