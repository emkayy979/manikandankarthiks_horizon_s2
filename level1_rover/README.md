Rover motion Calculator

This is a python based program to calculate the time taken for a rover to move from one point to another using motion parametres given by the user (initial velocity, final/max velocity and acceleration)

How it works:

> Finds the total distance to be covered using distance formula.
> Check if the rover will reach its max speed before or after reaching its destination.
> find total time taken according to the specific case.
>Error Handling: 
The program uses try and except blocks to check for value errors. (using wrong data type as input)

Environment Setup:
Requirement: Python 3.13
Library: math (Standard library, no installation required)

How to run:
>Navigate into the project folder:

Bash
>cd level1_rover

Execute the script:

Bash
>python level1rover_calc.py
